## Track 2: Encrypt / Ika (PRIMARY)

**Listing**: https://superteam.fun/earn/listing/encrypt-ika-frontier-april-2026  
**Prize**: $15,000 USDC (1st: $10K / 2nd: $3K / 3rd: $1K / 4th-5th: $500 each)  
**Winner Announcement**: June 1, 2026  
**Eligibility**: Global  
**Theme**: Bridgeless Capital Markets + Encrypted Capital Markets  

### Why AXIOM Wins This

AXIOM is the flagship use case for Ika's bridgeless capital markets thesis:

1. **dWallets are architecturally central** — not bolted on. Every AI agent action in AXIOM goes through an Ika dWallet. This is the only way to make AI-managed DeFi safe.

2. **Bridgeless cross-chain collateral** — users post native BTC or ETH as collateral without wrapping or bridging. This is exactly what Ika was built for, applied to a real lending market.

3. **Encrypted capital markets** — ZK credit proofs are "encrypted capital" in the sense Ika intends. Your creditworthiness is proven cryptographically without exposing your financial data.

4. **Agent policy enforcement** — AXIOM demonstrates Ika's key value proposition: AI agents with bounded execution. The agent can't steal funds. The policy is law.

### Submission Pitch for Ika Track

> "AXIOM demonstrates both Ika use cases simultaneously. Bridgeless: borrowers use native BTC/ETH as collateral on Solana without bridges — Ika controls the cross-chain asset directly via dWallet policy. Encrypted: credit scoring is done by a local QVAC model whose output is proven via ZK proof — encrypted creditworthiness. Every AI agent action in AXIOM is policy-constrained by Ika dWallets. This solves the hardest unsolved problem in AI-managed DeFi."

GoldRush is the data backbone of AXIOM's credit scoring system. The QVAC credit agent pulls cross-chain wallet history via GoldRush APIs — decoded DeFi interactions, USDT transfer patterns, historical portfolio data — to build a rich feature vector for the local credit model.

GoldRush x402 (their agent-optimized API) is used specifically because it returns structured data designed for LLM consumption — a perfect fit for QVAC agents.

### Submission Pitch for GoldRush Track

> "GoldRush is the eyes of AXIOM's credit agent. We use it for wallet age, DeFi interaction history, USDT holding patterns, LP position history, and cross-chain balance data. Remove GoldRush, and the credit model loses 70% of its signal quality."
